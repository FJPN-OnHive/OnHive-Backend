### e-hive - orders

orders / carts Back-End

#### Orders Flux:


![order flux](https://ehive-apidocs-assets.s3.us-east-2.amazonaws.com/Order_Flux.png)

##### Endpoints:

* **Criar Pedido por SKU** - Orders -> CrateOrderBySku
* **Buscar informaçõies dos provedores de pagamento** - Payments -> GetProvidersInfo
* **Checkout** - Payments -> Checkout
* **Cancelar Pedido** - Orders -> CancelOrder
* **Solicitar reembolso** - Orders -> RequestRefoundOfOrder

#### Cart Flux:

![order flux](https://ehive-apidocs-assets.s3.us-east-2.amazonaws.com/Cart_Flux.png)

##### Endpoints:

* **Criar Carrinho por SKU** - Carts -> CrateCartBySku
* **Adicionar Itens ao Carrinho** - Carts -> AddProductToCart
* **Modificar Produto do Carrinho** - Carts -> ChangeProductQuantity
* **Remover Itens do Carrinho** - Carts -> RemoveProductFromCart
* **Deletar Carrinho** - Carts -> DeleteCart
* **Criar Pedido apartir do Carrinho** - Orders -> CreateOrderFromCart


##### Observações sobre Cart (carrinho) 

* Quando for adicionado um produto que já exista no carrinho sera adicionado a quantidade desse produto.
* Quando um produto tem sua quantidade mopdificada para 0, o memso será removido do carrinho.
*  Quando o ultimo produto é removido o carrinho é deletado.
*  Quando uma Order for criada apartir de um carrinho este carrinho será deletado.
